# 🚀 PUGC Smart Assistant Platform (AI + University Automation System)

A full-stack AI-powered SaaS-style university automation platform designed to streamline student services using chatbot intelligence, LLM integration, and automated form generation systems.

---

## 🧠 Project Overview

PUGC Smart Assistant Platform is an intelligent university service system that combines AI chatbot capabilities with automation tools to assist students and administrative staff.

It supports:
- Student queries
- Admin management
- Notifications
- Analytics dashboard
- Subscription-based system design

It goes beyond a traditional chatbot by integrating:
- LLM-based responses
- Automated form generation (Challan & Application forms)
- Role-based dashboards
- Analytics and reporting system
